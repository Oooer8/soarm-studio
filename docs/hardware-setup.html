Hardware setup

在开始标定前，我们需要先查找好机械臂和相机的串口，绑定并且设置好配置。

这一步前置工作：

  • 机械臂所有电机编好号
  • 组装好机械臂，并且连接好电源和信号线
  • 连接好摄像头信号线
  • 布置好操作场景

扫描硬件

扫描当前机器可见的机械臂串口和 USB 摄像头，只保留后续绑定需要关注的字段。

CLI 
soarm-studio scan --include-system

机械臂 setup

  1. Probe arms

    两个串口都出现时，先确认每个端口是否能读到机械臂电机。

    CLI 
    soarm-studio scan --probe-arms
    查看精简结果 
    {
  "summary": {
    "ok": true,
    "probed_ports": 2,
    "online_ports": 2
  },
  "arm_ports": [
    {
      "device": "/dev/cu.usbmodem5A7C1190351",
      "ok": true,
      "expected_ids": [1, 2, 3, 4, 5, 6],
      "online_ids": [1, 2, 3, 4, 5, 6]
    },
    {
      "device": "/dev/cu.usbmodem5A7C1219051",
      "ok": true,
      "expected_ids": [1, 2, 3, 4, 5, 6],
      "online_ids": [1, 2, 3, 4, 5, 6]
    }
  ],
  "next_steps": [
    "Use the verified devices in setup arms as leader/follower ports."
  ]
}

    Probe 阶段只确认端口上是否能读到标准 SOARM 电机，默认使用电机 ID 1-6 和 SDK 默认波特率；如果你改过电机编号，用 --ids 指定。

  2. Setup arm roles

    将确认过的 leader/follower 串口写入本机运行配置。

    CLI 
    soarm-studio setup arms \
  --leader-port /dev/cu.usbmodem5A7C1219051 \
  --follower-port /dev/cu.usbmodem5A7C1190351
    查看精简结果 
    {
  "session_config": "configs/session.yaml",
  "arm_configs": {
    "leader": "configs/arms/leader.yaml",
    "follower": "configs/arms/follower.yaml"
  },
  "warnings": []
}

摄像头 setup

  1. Preview cameras

    保存预览图，用图片内容确认 OpenCV index 和物理摄像头位置。

    CLI 
    soarm-studio scan --preview-cameras \
  --camera-indices 0,1,2,3 \
  --backend avfoundation \
  --output-dir previews/cameras
    查看精简结果 
    {
  "summary": {
    "ok": true,
    "previewed": 2,
    "saved_frames": 2
  },
  "camera_previews": [
    {
      "index": 0,
      "ok": true,
      "backend": "avfoundation",
      "path": "previews/cameras/camera_0.jpg",
      "width": 640,
      "height": 480
    },
    {
      "index": 1,
      "ok": true,
      "backend": "avfoundation",
      "path": "previews/cameras/camera_1.jpg",
      "width": 640,
      "height": 480
    }
  ],
  "next_steps": [
    "Open the saved preview images, then use the confirmed indexes in setup cameras."
  ]
}

  2. Setup camera roles

    将确认过的 wrist 和 third-person 摄像头 index 写入配置。

    CLI 
    soarm-studio setup cameras \
  --backend avfoundation \
  --wrist-index 1 \
  --third-person-index 0
    查看精简结果 
    {
  "session_config": "configs/session.yaml",
  "assigned": {
    "wrist": {
      "device": 1,
      "backend": "avfoundation",
      "width": 640,
      "height": 480,
      "fps": 30
    },
    "third_person": {
      "device": 0,
      "backend": "avfoundation",
      "width": 640,
      "height": 480,
      "fps": 30
    }
  }
}

检查配置

验证机械臂和摄像头绑定是否能被当前配置识别，并运行上线前检查。

CLI 
soarm-studio check --config configs/session.yaml --overwrite
查看精简结果 
{
  "bindings_ok": true,
  "bindings": {
    "leader": {
      "device": "/dev/cu.usbmodem5A7C1190351",
      "ok": true
    },
    "follower": {
      "device": "/dev/cu.usbmodem5A7C1219051",
      "ok": true
    },
    "camera:wrist": {
      "device": 1,
      "ok": true
    },
    "camera:third_person": {
      "device": 0,
      "ok": true
    }
  },
  "preflight": {
    "ok": true,
    "errors": [],
    "warnings": []
  }
}
