Step 1 Hardware setup

Step 1 discovers local hardware, assigns roles, and writes the runtime config. It stops before calibration or motion.

Scan hardware

List serial ports and USB camera inventory.

soarm-studio scan --include-system

Probe arms

Probe one bus at a time when ports are ambiguous.

soarm-studio scan --probe-arms --arm-config ../soarm-sdk/configs/soarm.yaml

Setup arm roles

Save explicit leader and follower ports.

soarm-studio setup arms \
  --leader-port /dev/cu.usbmodemLEADER \
  --follower-port /dev/cu.usbmodemFOLLOWER \
  --base-arm-config ../soarm-sdk/configs/soarm.yaml

Preview cameras

Inspect saved frames before assigning wrist and third-person.

soarm-studio scan --preview-cameras \
  --camera-indices 0,1,2,3 \
  --backend avfoundation \
  --output-dir previews/cameras

Setup camera roles

Save indexes confirmed from preview images.

soarm-studio setup cameras \
  --backend avfoundation \
  --wrist-index 1 \
  --third-person-index 0

Check readiness

Verify bindings and run live readiness checks.

soarm-studio check --config configs/session.yaml --overwrite
