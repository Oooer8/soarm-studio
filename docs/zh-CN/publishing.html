发布到 GitHub Pages

这个文档站是静态的。源文件位于 main 分支的 docs/，GitHub Actions 会将该目录发布为 Pages artifact。

  1. 打开 GitHub 仓库设置: Oooer8/soarm-studio。
  2. 在 Pages 中选择 GitHub Actions 作为源。
  3. .github/workflows/pages.yml 中的 workflow 会上传 docs/。
  4. deploy job 会把 artifact 发布到 github-pages 环境。

后续照片、截图和视频应放在 docs/assets/。摄像头预览、数据集、标定采集等运行时输出默认保留在文档目录之外，除非有意加入整理后的副本。
