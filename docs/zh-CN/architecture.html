架构

SOARM Studio 有意保持很薄。它负责工作流编排、本地角色分配、摄像头预览、录制 sidecar 和一个小型 Web 界面。电机 I/O、运动、安全和标定仍属于 soarm-sdk。

运行路径 text 
leader arm -> 30 Hz teleop loop -> SDK direct joint stream -> follower arm
                         latest-frame cameras -> synchronized samples -> LeRobot-v3-compatible files
模块 职责
hardware.ports 串口发现和 SOARM 总线探测。
hardware.cameras USB 摄像头清单、OpenCV 预览、阻塞式摄像头适配器，以及用于非阻塞遥操作采集的 latest-frame 封装。
assignment 写入本地主臂/从臂以及 wrist/third-person 角色。
hardware.runtime 拥有用于 check、teleop 和 record 的已连接机械臂与摄像头。
recording 运行 episode 并写入本地数据集 sidecar。
datasets 兼容 LeRobot v3 的本地写入器和检查工具。
