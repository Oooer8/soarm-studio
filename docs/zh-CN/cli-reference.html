CLI 参考

公开 CLI 有意保持精简。

命令 使用
scan 查找串口、摄像头以及可选预览；热插拔前后重复运行 --probe-arms，通过可访问总线列表的变化确认主从端口。
setup 把机械臂和摄像头角色保存到运行配置。
check 验证绑定和就绪状态。
calibrate 运行交互式零位和连续范围采集。
teleop 运行主从臂控制并打印循环指标。
record 录制本地 episode 和 sidecar。
dataset 检查、验证或导出本地数据集。
web 运行本地浏览器界面。
