SOARM Studio 文档

SOARM Studio 是双 SOARM 启动、标定、遥操作和本地数据采集的应用层。它保持用户流程清晰，同时把底层机械臂行为委托给 soarm-sdk。

README 与文档

README 是简短的项目入口。这些文档是更详细的操作手册，涵盖安装、概念、硬件设置、标定、录制、媒体示例和故障排查。

推荐命令路径

使用 scan， setup， check， calibrate， teleop，以及 record。旧的重复命令形态已经从公开 CLI 中移除。

一个生成的运行配置
