配置

setup 会写入一个运行配置和两份机械臂配置。这些文件会被忽略，因为它们依赖本机 USB 端口、摄像头 index 和标定后的限位。

生成文件 text 
configs/session.yaml
configs/arms/leader.yaml
configs/arms/follower.yaml
!
不要把示例配置当成运行状态

configs/sessions/ 下的文件只是模板；运行 setup 后，真实运行使用 configs/session.yaml。
