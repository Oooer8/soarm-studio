SOARM Studio documentation

SOARM Studio is the application layer for dual SOARM bring-up, calibration, teleoperation, and local data collection. It keeps the user workflow clear while delegating low-level arm behavior to soarm-sdk.

i
README vs docs

The README is a short project entry point. These docs are the detailed operating manual for installation, concepts, hardware setup, calibration, recording, media examples, and troubleshooting.

Recommended command path

Use scan, setup, check, calibrate, teleop, and record. Older duplicate command shapes were removed from the public CLI.

One generated runtime config

configs/session.yaml is the local runtime config for one machine. Example configs remain templates under configs/sessions/.
