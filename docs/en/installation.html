Installation

The conda environment installs SOARM Studio in editable mode and pulls soarm-sdk from the GitHub dependency declared by the hardware extra.

Install the environment zsh 
conda env create -f environment.yml
conda activate soarm-studio
Validate the local install zsh 
soarm-studio --help
soarm-studio check --config configs/sessions/mock.yaml --overwrite
