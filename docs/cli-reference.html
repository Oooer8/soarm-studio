CLI reference

The public CLI is intentionally small.

Command Use
scan Find ports, cameras, and optional previews.
setup Save arm and camera roles into runtime config.
check Verify bindings and readiness.
calibrate Run interactive zero and continuous range capture.
teleop Run leader-to-follower control.
record Record local episodes and sidecars.
dataset Inspect, validate, or export local datasets.
web Run the local browser UI.
